1 COMPROBACIÓN ENTORNO

node -v
npm -v


2 INSTALACIÓN DE angular

npm install -g @angular/cli@19

3 FUNCIONANDO

Crear proyecto - ng new <nombre_proyecto>
Arrancarlo - ng serve -o 
Añadimos bootsrap - npm install bootstrap
Añadiendo componentes - ng g c <ruta_componente/nombre>


Añadiendo clase IMC - ng g class models/imc
Add el enum - ng g enum models/tipoImc




crear el componente (plantilla, css y clase) ng g c components/alumn
enrutar el componente - {path:'alumnos', component: AlumnoComponent} en  app routes
definir el tipo de datos (DTO)
crear el servicio - ng g service services/alumno
configurar el acceso HTTP - HttpModule - provideHttpClient() en app config ts
inyectar el servicio en el componente
